# Decisões Tomadas — Consolidação do System Prompt Clarify

## Decisão 1: Unificação dos Perfis de Liderança (Técnico, Transição, Engajado) e Modelo SBI
- **Contexto**: A IA precisa saber lidar com o público de gestores da ClearIT que possui diferentes maturidades.
- **Decisão**: Configurar as diretrizes de adaptabilidade de tom juntamente com a pauta clássica de 5 blocos de 1:1 e modelo de feedback SBI.
- **Justificativa**: Atende de forma customizada cada perfil de liderança sem desrespeitar os padrões de banco de dados e dados estruturados já estabelecidos.

## Decisão 2: Listagem das 8 Competências no Prompt de Sistema
- **Contexto**: Para estruturar o PDI na Opção 4, a IA precisa de referências explícitas das competências válidas.
- **Decisão**: Mapear as 8 competências oficiais diretamente nas regras de PDI do prompt.
- **Justificativa**: Melhora a precisão semântica das sugestões de objetivos de PDI e previne o cadastro de dados errôneos.
