# Contexto — Matriz de Competências Oficial ClearIT

## Situação Inicial
Anteriormente, havíamos estruturado a KB do Framework de Levels com 5 competências genéricas de exemplo. O usuário compartilhou o código fonte da página HTML contendo a matriz de competências real da ClearIT com 8 competências formais e seus respectivos descritores de performance acima/abaixo do esperado.

## Motivação
Garantir o alinhamento completo do produto e do banco de dados com a matriz de competências organizacional real da ClearIT. Isso blinda o mapeamento de PDI no banco de dados Express/Prisma e na renderização do React contra erros de domínio.

## Restrições
- Copiar fielmente os termos e os descritores do HTML fornecido.
