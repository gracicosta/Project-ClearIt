# Criação do Agente de IA Clarify — 2026-07-04

## 🎯 Objetivo
Estruturar o diretório para prompts de inteligência artificial e definir o prompt de sistema do agente Clarify, alinhado com as especificações de negócio e regras de compliance.

## 📊 Resultados
- Criação da pasta de agentes em `agents/clarify/`.
- Criação do prompt de sistema em `system_prompt.md` com suporte completo para as features de 1:1, feedbacks SBI, PDI simplificado e LGPD.
- Adição da funcionalidade F-07 para análise e auditoria de feedbacks históricos em PDF, além do menu inicial de boas-vindas do agente.
