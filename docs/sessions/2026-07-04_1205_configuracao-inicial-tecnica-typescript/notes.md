# Notas e Observações — Configuração Inicial Técnica (TypeScript)

## Insights e Aprendizados
- TypeScript e Prisma formam uma excelente dupla para iniciantes devido ao auto-complete nativo gerado a partir do banco de dados, reduzindo erros de digitação e melhorando a produtividade.
- O uso de `tsx` no backend agiliza a inicialização e evita atritos com compiladores de TypeScript.
